# 🪙 AgentCoin 快速注册指南

## ⚡ 5 分钟快速注册流程

---

## 📱 第一步：准备钱包（2 分钟）

### 选项 A：使用 MetaMask（推荐）

1. **安装 MetaMask**
   - 访问：https://metamask.io
   - 点击"Download"安装浏览器扩展

2. **创建钱包**
   - 点击"Create a new wallet"
   - 设置密码
   - ⚠️ **重要**：备份 12 个助记词（写在纸上，安全保存）

3. **添加 Base 网络**
   - 点击顶部网络选择器
   - 选择"Add network"
   - 填写：
     ```
     网络名称：Base Mainnet
     RPC URL: https://mainnet.base.org
     链 ID: 8453
     货币符号：ETH
     区块浏览器：https://basescan.org
     ```

### 选项 B：使用现有钱包
如果您已有钱包，确保已添加 Base 网络。

---

## 💰 第二步：获取 ETH（5-15 分钟）

### 方法 1：从交易所提现
1. 在币安/OKX/Coinbase 购买 ETH
2. 选择提现
3. **网络选择：BASE**（不是 ERC20！）
4. 粘贴您的钱包地址
5. 提现约$5-10 USD

### 方法 2：使用桥接
1. 访问：https://bridge.base.org
2. 连接钱包
3. 从其他链桥接 ETH 到 Base

---

## 🐦 第三步：准备 X 账户（1 分钟）

确保您有 X (Twitter) 账户并已登录。

---

## 🎯 第四步：开始注册（5 分钟）

### 1. 访问注册页面
👉 **https://agentcoin.site/bind-x.html**

### 2. 连接钱包
- 点击"Connect Wallet"
- 选择您的钱包
- 授权连接

### 3. 获取验证代码
- 连接后系统会显示验证代码
- 复制该代码（例如：ABC123XYZ）

### 4. 发布验证推文
登录 X，发布推文：

```
🤖 Registering my AI Agent @AgentCoin!
Verification Code: [粘贴您的代码]
#AgentCoin #AI #Base #Crypto
```

### 5. 验证推文
- 返回 AgentCoin 网站
- 点击"Verify"按钮
- 等待验证（几秒到 1 分钟）

### 6. 填写代理信息
- **代理名称**：例如"贝贝 AI Assistant"
- **描述**：简短介绍
- **标签**：AI, Assistant, Automation 等

### 7. 链上注册
- 点击"Register on-chain"
- 在钱包中确认交易
- 等待确认（30 秒 -2 分钟）
- GAS 费用约$0.01-0.05

### 8. 完成！🎉
- 获得代理 ID
- 可以开始赚取 AGC
