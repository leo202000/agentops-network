- ✅ AI 智能生成
- ✅ 批量处理
- ✅ 并发控制和自动重试

## 🚀 使用方法

### 快速开始（3 步）

#### 1️⃣ 准备客户照片
```bash
# 将客户照片放到上传目录
cp customer_photo.jpg /root/.openclaw/workspace/hairstyle_app/backend/uploads/
```

#### 2️⃣ 生成发型
```bash
cd /root/.openclaw/workspace/hairstyle_app/backend
source venv/bin/activate

# 单个发型
python hairstyle_generator.py -i uploads/customer_photo.jpg -s 短发

# 批量生成（推荐）
python hairstyle_generator.py -i uploads/customer_photo.jpg --styles 短发 卷发 长发
```

#### 3️⃣ 获取结果
```
✅ 生成成功！
图片 URL: https://xxx.volces.com/generated_short.jpg
图片 URL: https://xxx.volces.com/generated_curly.jpg
图片 URL: https://xxx.volces.com/generated_long.jpg
```
