## 🔎 问题分析

### 50400 错误可能原因

1. **服务未正确开通**
   - 可能只开通了"视觉服务"，未开通"即梦 AI"
   - 即梦可能是独立产品，需要单独开通

2. **API Key 不匹配**
   - 即梦可能使用独立的 API Key
   - 不是视觉服务的通用 Key

3. **应用分组问题**
   - 火山引擎需要创建"应用"才能调用 API
   - 需要在应用中绑定服务

4. **服务端点错误**
   - 即梦可能有独立的 API 端点
   - 不是 `visual.volcengineapi.com`

---

## ✅ 解决方案

### 步骤 1: 检查即梦控制台

访问：https://console.volcengine.com/jimeng

或：控制台 → 人工智能 → 即梦 AI

### 步骤 2: 创建应用

1. 在即梦控制台中创建"应用"
2. 获取应用的 API Key 和 Secret Key
3. 绑定"图片生成 4.0"服务

### 步骤 3: 检查服务绑定

1. 进入应用详情
2. 查看已绑定的服务
3. 确保包含"图片生成"或"即梦"

### 步骤 4: 使用正确的 API 端点

即梦可能的 API 端点：
- `jimeng.volcengineapi.com`
- `api.jimeng.ai`
- 或其他独立端点
