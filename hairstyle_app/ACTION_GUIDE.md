| B 站 | 周更 2 条 | 长视频 | 品牌背书 |

### 内容配方

**爆款对比图**（40%）：
```
左图：原图（普通发型）
右图：AI 生成（改造后）
标题："圆脸女生换了这个发型，同事都以为去整容了！"
```

**干货知识**（20%）：
```
标题："5 种脸型适合的发型大全，建议收藏！"
内容：脸型判断 + 发型推荐 + AI 示例
```

**真人出镜**（30%）：
```
形式：设计师讲解 + AI 演示
主题："为什么你的发型总显老？3 个误区你中了几个"
```

**互动话题**（10%）：
```
标题："猜猜这个发型适合什么脸型？"
目的：提高评论率
```

## 💰 产品交付 SOP

### 99 元 AI 设计报告

**交付内容**：
1. 3 款发型设计方案（AI 生成图）
2. 脸型分析报告
3. 发型打理建议
4. 7 天售后服务

**交付流程**：
```
用户下单 → 发送照片 → AI 生成 3 款 → 人工审核 → 发送报告 → 售后跟进
   ↓           ↓           ↓          ↓          ↓          ↓
 自动       1 小时内    15 分钟     30 分钟    即时      7 天
```

**质量标准**：
- [ ] 五官无变形
- [ ] 发型边缘自然
- [ ] 脸型分析准确
- [ ] 建议可执行

**重做标准**：
- 用户不满意可免费重做 1 次
- 重做仍不满意可退款 50%
