## 📋 核心优化

### 1. 提示词优化

**优化前**（通用）:
```
保持人物脸部完全一致，只改变发型为齐肩发，shoulder length bob...
```

**优化后**（真人专用）:
```
【人物发型变换】
保持人物面部特征完全一致，只改变发型。
原始特征：保持五官、脸型、肤色、表情不变。
目标发型：shoulder length bob hairstyle, classic and timeless...
风格要求：realistic photo, high quality, professional photography, natural lighting, detailed hair texture
负面提示：保持人物身份特征，不要改变面部结构
```

**优化点**:
- ✅ 明确指定"人物发型变换"
- ✅ 强调保持面部特征
- ✅ 添加详细风格要求
- ✅ 包含负面提示词

---

### 2. 人脸保护参数

```python
extra_body={
    "image": image_url,
    "watermark": False,
    # 人脸保护参数
    "face_preserve": True,  # 保持人脸
    "strength": 0.7,        # 重绘强度
}
```

**参数说明**:
| 参数 | 推荐值 | 说明 |
|------|--------|------|
| `face_preserve` | `True` | 启用人脸保护 |
| `strength` | `0.6-0.7` | 重绘强度（0.3-0.9） |

---

### 3. 重绘强度建议

| 强度值 | 效果 | 适用场景 |
|--------|------|----------|
| 0.3-0.4 | 轻微变换 | 微调发型细节 |
| 0.5-0.6 | 中等变换 | 相似发型变换 |
| **0.7** | **标准变换** | **常规发型变换（推荐）** |
| 0.8-0.9 | 彻底变换 | 完全不同的发型 |

## 📸 输入图片要求

### 推荐规格

| 要求 | 说明 |
|------|------|
| **格式** | JPG, PNG, WebP |
| **尺寸** | 512x512 - 2048x2048 |
| **人脸** | 清晰可见，正脸或侧脸≤45° |
| **光线** | 均匀照明，避免强阴影 |
| **背景** | 简洁背景为佳（非必须） |

### 示例图片

✅ **推荐**:
- 正面或微侧脸照片
- 光线均匀
- 人脸清晰
- 原发型可见

❌ **不推荐**:
- 侧脸角度过大（>45°）
- 强阴影或逆光
- 人脸被遮挡（头发、眼镜、口罩等）
- 图片模糊

## 🚀 使用示例

### 基础使用

```python
from hairstyle_generator_v3 import HairstyleGeneratorV3

# 初始化生成器
generator = HairstyleGeneratorV3()

# 生成发型
result = generator.generate(
    image_url="https://example.com/user_photo.jpg",
    style="齐肩发",
    strength=0.7  # 标准强度
)

if result['success']:
    print(f"生成成功：{result['image_url']}")
    print(f"耗时：{result['elapsed']:.1f}秒")
else:
    print(f"生成失败：{result['error']}")
```

### 批量生成

```python
# 批量生成多种发型
styles = ["齐肩发", "梨花头", "丸子头", "空气刘海"]

results = generator.generate_batch(
    image_url="https://example.com/user_photo.jpg",
    styles=styles,
    interval=2  # 请求间隔 2 秒
)

# 显示结果
for result in results:
    if result['success']:
        print(f"✅ {result['style']}: {result['image_url']}")
```

### 自定义强度

```python
# 轻微变换（保留更多原发型特征）
result_light = generator.generate(
    image_url="https://example.com/user_photo.jpg",
    style="波浪卷",
    strength=0.5  # 轻微变换
)

# 彻底变换（完全不同的发型）
result_strong = generator.generate(
    image_url="https://example.com/user_photo.jpg",
    style="短发",
    strength=0.8  # 彻底变换
)
```

## ⚠️ 注意事项

### API 使用

1. **请求间隔**: 建议 2 秒，避免触发限流
2. **并发限制**: 单次批量建议≤10 种发型
3. **超时处理**: 单次生成约 10-20 秒

### 图片处理

1. **图片上传**: 需先上传到 TOS 或其他公网存储
2. **图片 URL**: 必须是公网可访问的 URL
3. **图片大小**: 建议≤5MB

### 效果优化

1. **原发型**: 原发型与目标发型差异越大，效果可能越明显
2. **发色**: 染发类发型受原发色影响
3. **发量**: 发量过少可能影响某些发型效果

## 🔧 故障排除

### 常见问题

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 生成失败 | 图片 URL 无法访问 | 检查 URL 是否公网可访问 |
| 人脸变形 | 重绘强度过高 | 降低 strength 到 0.5-0.6 |
| 发型不变 | 重绘强度过低 | 提高 strength 到 0.7-0.8 |
| API 限流 | 请求过于频繁 | 增加请求间隔到 3-5 秒 |
| 超时 | 网络问题 | 检查网络连接，重试 |

### 错误代码

| 错误代码 | 说明 | 解决方案 |
|----------|------|----------|
| 400 | 请求参数错误 | 检查图片和参数格式 |
| 401 | API Key 无效 | 检查 ARK_API_KEY 配置 |
| 429 | 请求限流 | 降低请求频率 |
| 500 | 服务器错误 | 稍后重试 |
