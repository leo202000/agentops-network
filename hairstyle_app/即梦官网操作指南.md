# 即梦官网操作指南 - 抓取参数

**目标**: 在即梦官网生成大波浪发型，截图参数面板分享给我们

---

## 📱 操作步骤

### 步骤 1: 访问即梦官网

**网址**: https://jimeng.jianying.com/ai-tool/home

**操作**:
1. 打开浏览器 (Chrome/Edge/Safari)
2. 访问上述网址
3. 登录账号 (如果没有需要注册)

---

### 步骤 2: 选择图生图功能

**界面导航**:
```
首页 → AI 工具 → 图生图
或
首页 → 立即创作 → 图生图 3.0/4.0
```

**注意**:
- 找到"图生图"或"Image-to-Image"功能
- 确认使用的是 **seed3l_single_ip** 或 **doubao-seedream-4.0** 模型

---

### 步骤 3: 上传照片

**操作**:
1. 点击"上传图片"按钮
2. 选择照片文件：`file_43---551c649a-c68f-4a52-807d-fb458d5f5136.jpg`
3. 等待上传完成

**照片位置**:
- 如果照片在手机上：通过微信/QQ 传到电脑
- 如果照片在电脑上：直接从文件夹选择

---

### 步骤 4: 输入提示词

**Prompt (正向提示词)**:
```
大波浪卷发，浓密大波浪，long wavy curly hair, voluminous waves, glamorous curls
```

**Negative Prompt (负面提示词)**:
```
straight hair, short hair, 直发，短发
```
*(如果官网支持负面 prompt)*

---

### 步骤 5: 调整参数 ⭐ (最关键!)

**需要找到的参数面板**:

通常在界面右侧或底部，寻找以下参数：

| 参数名 | 可能显示为 | 我们的目标值 |
|--------|-----------|-------------|
| **重绘强度** | strength / 重绘幅度 / Denoising strength | ? (待发现) |
| **提示词相关性** | cfg_scale / CFG Scale / 提示词引导 | ? (待发现) |
| **采样步数** | sample_steps / Steps / 采样迭代数 | ? (待发现) |
| **模型版本** | model / 模型 | seed3l_single_ip 或 4.0 |
| **负面提示词** | negative_prompt / 负面提示词 | ? (待发现) |

---

### 步骤 6: 生成图片

**操作**:
1. 点击"生成"或"立即创作"按钮
2. 等待生成完成 (通常 10-30 秒)
3. 查看生成结果

---

### 步骤 7: 截图参数面板 ⭐⭐⭐ (最重要!)

**需要截图的内容**:

**截图 1: 参数设置面板**
- 包含所有参数滑块/输入框
- 显示当前参数值
- 确保清晰可见

**截图 2: 高级设置** (如果有)
- 点击"高级设置"或"更多参数"
- 截图显示的所有额外参数

**截图 3: 生成结果** (可选)
- 生成的图片效果
- 用于对比我们的 API 生成效果

**截图方法**:

**Windows**:
- `Win + Shift + S` - 区域截图
- `PrintScreen` - 全屏截图
- 截图工具 - 搜索"截图工具"

**macOS**:
- `Cmd + Shift + 4` - 区域截图
- `Cmd + Shift + 3` - 全屏截图

**手机**:
- 音量下 + 电源键 (Android)
- 侧边按钮 + 音量上 (iPhone)

---

## ⚠️ 常见问题

### Q1: 找不到参数滑块？
**A**: 寻找以下按钮:
- "高级设置"
- "更多参数"
- "⚙️ 设置"图标
- "参数配置"

### Q2: 参数名称不一样？
**A**: 常见别名:
- strength = 重绘强度 = 重绘幅度 = Denoising strength
- cfg_scale = CFG Scale = 提示词相关性 = 提示词引导
- sample_steps = 采样步数 = Steps = 迭代次数

### Q3: 官网不支持某些参数？
**A**: 记录实际情况:
- 哪些参数可以调整
- 哪些参数是固定的
- 默认值是多少

### Q4: 生成失败/报错？
**A**: 截图错误信息，分享给我们分析

### Q5: 需要付费/会员？
**A**: 记录:
- 免费额度有多少
- 是否需要充值
- 会员价格
