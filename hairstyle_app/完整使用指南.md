1. **客户上传照片** - 支持 JPG/PNG 格式
2. **选择发型风格** - 13 种预设发型
3. **AI 生成** - 火山引擎即梦 API
4. **输出结果** - 新发型效果图

| 类别 | 发型 |
|------|------|
| 基础 | 短发、卷发、长发、直发 |
| 造型 | 马尾、辫子、波浪卷、大波浪 |
| 刘海 | 中分、斜刘海 |
| 染发 | 染发红、染现金、染发棕 |

## 🚀 快速开始

### 步骤 1: 确认配置

```bash
# 检查 API 密钥
cat /root/.openclaw/workspace/.env | grep JIMENG

# JIMENG_ACCESS_KEY_ID=AKLTYmUyMTA5MzYzM2Nm...
# JIMENG_SECRET_ACCESS_KEY=T0RZeU1XWTJZV1Uw...
```

### 步骤 2: 准备客户照片

```bash
# 照片要求:
# - 格式：JPG 或 PNG
# - 大小：< 5MB
# - 分辨率：建议 512x512 以上
# - 内容：清晰的人脸照片

# 示例：将客户照片放到工作目录
cp /path/to/customer_photo.jpg /root/.openclaw/workspace/hairstyle_app/backend/uploads/
```

### 步骤 3: 生成发型

#### 方式 A: 命令行（推荐）

```bash
cd /root/.openclaw/workspace/hairstyle_app/backend
source venv/bin/activate

# 生成单个发型
python hairstyle_generator.py -i uploads/customer_photo.jpg -s 短发

# 批量生成（3 个发型）
python hairstyle_generator.py -i uploads/customer_photo.jpg --styles 短发 卷发 长发

# 查看全部发型选项
python hairstyle_generator.py --list-styles
```

#### 方式 B: Python 代码

```python
from hairstyle_generator import HairstyleGenerator
import os

# 初始化生成器
generator = HairstyleGenerator(
    access_key=os.getenv("JIMENG_ACCESS_KEY_ID"),
    secret_key=os.getenv("JIMENG_SECRET_ACCESS_KEY")
)

# 生成单个发型
result = generator.generate(
    image_path="uploads/customer_photo.jpg",
    style="短发",
    wait=True  # 等待完成
)

# 检查结果
if result["success"]:
    print("✅ 生成成功！")
    for img in result["images"]:
        print(f"图片 URL: {img['url']}")
else:
    print(f"❌ 生成失败：{result['error']}")

# 批量生成
results = generator.generate_batch(
    image_path="uploads/customer_photo.jpg",
    styles=["短发", "卷发", "长发", "马尾"],
    delay=2.0  # 每个请求间隔 2 秒
)

# 查看批量结果
for r in results:
    status = "✅" if r["success"] else "❌"
    print(f"{status} {r['style']}: {r.get('status', r.get('error'))}")
```

#### 方式 C: API 服务

```bash
# 启动 API 服务
cd /root/.openclaw/workspace/hairstyle_app/backend
source venv/bin/activate
uvicorn main:app --host 0.0.0.0 --port 8002

curl -X POST http://localhost:8002/upload \
  -F "file=@customer_photo.jpg"

# 返回: {"url": "http://localhost:8002/uploads/xxx.jpg"}

# 2. 生成发型
curl -X POST http://localhost:8002/generate-hairstyle \
  -H "Content-Type: application/json" \
  -d '{
    "user_image": "http://localhost:8002/uploads/xxx.jpg",
    "style": "短发"
  }'
```

## ⚠️ 注意事项

### 1. API 密钥安全
```bash
# ✅ 正确：存储在 .env 文件
chmod 600 /root/.openclaw/workspace/.env

# ❌ 错误：硬编码在代码中
# 永远不要这样做！
```

### 2. 图片 URL 合规
```bash
# ✅ 正确：使用国内 CDN/OSS
- 火山引擎 TOS
- 阿里云 OSS
- 腾讯云 COS

# ❌ 错误：使用国外 CDN
- picsum.photos
- fastly 域名
- 其他国外服务
```

### 3. 并发控制
```python
# ✅ 正确：串行处理 + 延迟
generator.generate_batch(
    styles=["短发", "卷发", "长发"],
    delay=2.0  # 每个请求间隔 2 秒
)

# ❌ 错误：并发请求
# 会导致 API 限流错误
```

## 🆘 故障排查

### 问题 1: API 密钥错误
```
错误：SignatureDoesNotMatch
解决：检查 .env 中的 AK/SK 是否正确
```

### 问题 2: 图片无法访问
```
错误：InvalidImageURL
解决：使用国内 OSS/TOS，确保公网可访问
```

### 问题 3: 并发超限
```
错误：Request Has Reached API Concurrent Limit
解决：已自动处理，增加 delay 参数
```

### 问题 4: 生成超时
```
错误：Timeout
解决：增加 timeout 参数（默认 180 秒）
```

### 问题 5: 脸部变化太大
```
解决：
1. 降低 strength 参数（0.4-0.5）
2. 在提示词强调"保持人物脸部完全一致"
3. 使用更具体的提示词
```

### 缓存策略
```python
# 对于相同的照片和发型，可以缓存结果
import hashlib

def get_cache_key(image_path, style):
    with open(image_path, 'rb') as f:
        image_hash = hashlib.md5(f.read()).hexdigest()
    return f"{image_hash}_{style}"
```
