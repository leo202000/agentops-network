**更新日期**: 2026-03-20  
**状态**: ✅ 已修复（根据即梦工程师反馈）

---

## ⚠️ 重要提示

根据火山引擎即梦工程师反馈：

> **入参链接必须是公网可以访问的资源链接，本地的链接是使用不了的，导致服务这边获取素材卡住了。**

### ❌ 错误示例
```python
# 本地链接，无法使用！
image_url = "http://localhost:8000/uploads/photo.jpg"
image_url = "http://127.0.0.1:8000/uploads/photo.jpg"
image_url = "file:///path/to/photo.jpg"
```

### ✅ 正确示例
```python
# 公网可访问的 URL
image_url = "https://bucket.tos-cn-beijing.volces.com/photo.jpg"
image_url = "https://bucket.oss-cn-hangzhou.aliyuncs.com/photo.jpg"

# 或 base64 编码
image_url = "data:image/jpeg;base64,/9j/4AAQSkZJRg..."
```

---

## 🔧 解决方案

系统已集成自动图片上传功能，支持三种方式：

### 方案 1: 火山引擎 TOS（推荐）⭐

**优势**:
- ✅ 同厂商，兼容性最好
- ✅ 国内 CDN，访问速度快
- ✅ 价格低廉

**配置步骤**:

1. 登录火山引擎 TOS 控制台：https://console.volcengine.com/tos
2. 创建存储桶（Bucket）
3. 获取访问密钥（AK/SK）
4. 设置环境变量：

```bash
# .env 文件
TOS_BUCKET=your-bucket-name
TOS_ACCESS_KEY=your_access_key
TOS_SECRET_KEY=your_secret_key
```

**使用示例**:
```python
from image_uploader import quick_upload

url = quick_upload("photo.jpg")
# 返回：https://bucket.tos-cn-beijing.volces.com/hairstyle/xxx.jpg
```

---

### 方案 2: 阿里云 OSS

**优势**:
- ✅ 稳定可靠
- ✅ 文档完善
- ✅ 广泛使用

**配置步骤**:

1. 登录阿里云 OSS 控制台：https://oss.console.aliyun.com
2. 创建 Bucket
3. 获取 AccessKey
4. 设置环境变量：

```bash
# .env 文件
OSS_BUCKET=your-bucket-name
OSS_ACCESS_KEY_ID=your_access_key_id
OSS_ACCESS_KEY_SECRET=your_access_key_secret
```

**使用示例**:
```python
from image_uploader import quick_upload

url = quick_upload("photo.jpg")
# 返回：https://bucket.oss-cn-hangzhou.aliyuncs.com/hairstyle/xxx.jpg
```

---

### 方案 3: Base64 编码（默认 Fallback）⭐

**优势**:
- ✅ 无需配置
- ✅ 立即可用
- ✅ 适合测试和小批量使用

**劣势**:
- ⚠️ 图片较大时传输慢
- ⚠️ 增加 API 请求体积

**使用方式**:
```python
# 不配置 TOS/OSS 时，自动使用 base64
from image_uploader import quick_upload

url = quick_upload("photo.jpg")
# 返回：data:image/jpeg;base64,/9j/4AAQSkZJRg...
```

**直接使用**:
```bash
python image_uploader.py photo.jpg
```

## 🔍 自动选择逻辑

系统会自动选择上传方式：

```
1. 检查 TOS 配置 → 如果存在，上传到 TOS
2. 检查 OSS 配置 → 如果存在，上传到 OSS
3. 默认 → 使用 base64 编码
```

**优先级**: TOS > OSS > Base64

## 🚀 快速开始

### 测试模式（Base64）

无需任何配置，直接使用：

```bash
cd /root/.openclaw/workspace/hairstyle_app/backend
source venv/bin/activate

python hairstyle_generator.py -i photo.jpg -s 短发
```

系统会自动将图片转换为 base64 并提交。

### 生产模式（TOS/OSS）

1. 配置环境变量（`.env` 文件）：

```bash
# 火山引擎 TOS（推荐）
TOS_BUCKET=my-hairstyle-bucket
TOS_ACCESS_KEY=AKLTY...
TOS_SECRET_KEY=T0RZeU...

# 或阿里云 OSS
OSS_BUCKET=my-hairstyle-bucket
OSS_ACCESS_KEY_ID=LTAI5t...
OSS_ACCESS_KEY_SECRET=...
```

2. 使用（自动上传到配置的对象存储）：

```bash
python hairstyle_generator.py -i photo.jpg -s 短发
```

## ⚠️ 常见问题

### Q1: 提示"本地链接无法使用"
**A**: 请使用 TOS/OSS 或 base64，不要用 localhost 链接

### Q2: TOS 上传失败
**A**: 检查：
- Bucket 名称是否正确
- AK/SK 是否正确
- 网络是否可访问 TOS

### Q3: Base64 太长导致 API 失败
**A**: 压缩图片或使用 TOS/OSS

### Q4: 图片太大上传慢
**A**: 建议：
- 压缩图片到 1MB 以内
- 使用 TOS/OSS 而不是 base64

## 🔐 安全提示

1. **AK/SK 安全**
   - ✅ 存储在 `.env` 文件
   - ✅ 设置权限：`chmod 600 .env`
   - ❌ 不要提交到 Git

2. **Bucket 权限**
   - ✅ 设置私有读写
   - ✅ 使用临时密钥（可选）
   - ❌ 不要公开写权限

3. **图片隐私**
   - ✅ 使用私有 Bucket
   - ✅ 生成临时访问 URL
   - ❌ 不要公开客户照片
