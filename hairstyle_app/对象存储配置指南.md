# 对象存储配置指南

**目标**: 配置对象存储以支持即梦 API 图生图功能

---

## 📋 为什么需要对象存储？

即梦 API 的图生图功能**不支持 base64 data URI 格式**，需要使用**公网可访问的图片 URL**。

对象存储可以提供：
- ✅ 公网可访问的图片 URL
- ✅ 稳定的存储服务
- ✅ 与火山引擎即梦同厂商（推荐 TOS）

---

## 🔧 方案选择

### 方案 A: 火山引擎 TOS（推荐）⭐

**优势**:
- 与即梦 API 同厂商，兼容性最好
- 国内访问速度快
- 价格相对便宜

**配置步骤**:

1. **注册火山引擎账号**
   - 访问：https://www.volcengine.com/
   - 注册/登录账号

2. **开通 TOS 服务**
   - 进入控制台：https://console.volcengine.com/tos/
   - 创建存储桶（Bucket）
   - 记录 Bucket 名称

3. **获取访问密钥**
   - 访问：https://console.volcengine.com/iam/
   - 创建访问密钥（AK/SK）
   - 保存 Access Key ID 和 Secret Access Key

4. **配置环境变量**
   ```bash
   # 编辑 .env 文件
   vim /root/.openclaw/workspace/.env
   
   # 添加以下配置
   TOS_BUCKET=your-bucket-name
   TOS_ACCESS_KEY=your-access-key-id
   TOS_SECRET_KEY=your-secret-access-key
   TOS_REGION=cn-beijing  # 根据 Bucket 所在区域选择
   ```

5. **安装 TOS SDK**
   ```bash
   pip install tos
   ```

---

### 方案 B: 阿里云 OSS

**优势**:
- 如果你已有阿里云账号
- 服务稳定可靠

**配置步骤**:

1. **注册阿里云账号**
   - 访问：https://www.aliyun.com/

2. **开通 OSS 服务**
   - 进入控制台：https://oss.console.aliyun.com/
   - 创建 Bucket
   - 记录 Bucket 名称

3. **获取访问密钥**
   - 访问：https://ram.console.aliyun.com/
   - 创建 AccessKey

4. **配置环境变量**
   ```bash
   # 编辑 .env 文件
   vim /root/.openclaw/workspace/.env
   
   # 添加以下配置
   OSS_BUCKET=your-bucket-name
   OSS_ACCESS_KEY_ID=your-access-key-id
   OSS_ACCESS_KEY_SECRET=your-access-key-secret
   OSS_REGION=oss-cn-hangzhou  # 根据 Bucket 所在区域选择
   ```

5. **安装 OSS SDK**
   ```bash
   pip install oss2
   ```

---

### 方案 C: 腾讯云 COS

**优势**:
- 如果你已有腾讯云账号

**配置步骤**:

1. **注册腾讯云账号**
   - 访问：https://cloud.tencent.com/

2. **开通 COS 服务**
   - 进入控制台：https://console.cloud.tencent.com/cos
   - 创建 Bucket

3. **获取访问密钥**
   - 访问：https://console.cloud.tencent.com/cam/capi
   - 创建密钥

4. **配置环境变量**
   ```bash
   # 编辑 .env 文件
   vim /root/.openclaw/workspace/.env
   
   # 添加以下配置
   COS_BUCKET=your-bucket-name
   COS_SECRET_ID=your-secret-id
   COS_SECRET_KEY=your-secret-key
   COS_REGION=ap-guangzhou  # 根据 Bucket 所在区域选择
   ```

5. **安装 COS SDK**
   ```bash
   pip install cos-python-sdk-v5
   ```

## 🔒 安全建议

1. **使用子账号密钥**
   - 不要使用主账号密钥
   - 为子账号分配最小权限（仅 TOS/OSS 读写）

2. **设置权限策略**
   ```json
   {
     "Version": "2012-10-17",
     "Statement": [
       {
         "Effect": "Allow",
         "Action": [
           "tos:PutObject",
           "tos:GetObject",
           "tos:DeleteObject"
         ],
         "Resource": ["arn:tos:*:*:bucket/your-bucket-name/*"]
       }
     ]
   }
   ```

3. **文件权限**
   ```bash
   chmod 600 /root/.openclaw/workspace/.env
   ```

4. **不要提交密钥到 Git**
   ```bash
   # 确保 .env 在 .gitignore 中
   echo ".env" >> .gitignore
   ```

## ✅ 验证配置

配置完成后，运行以下命令验证：

```bash
cd /root/.openclaw/workspace/hairstyle_app/backend
python3 -c "
import os
from pathlib import Path

env_path = Path('/root/.openclaw/workspace/.env')
if env_path.exists():
    with open(env_path, 'r') as f:
        content = f.read()
        if 'TOS_BUCKET' in content or 'OSS_BUCKET' in content:
            print('✅ 对象存储配置已找到')
        else:
            print('❌ 未找到对象存储配置')
else:
    print('❌ .env 文件不存在')
"
```
