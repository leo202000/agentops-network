## ⚠️ 异常处理流程

### 发现未删除的原图

**处理流程**：
1. 立即手动删除
2. 记录异常情况
3. 检查自动删除脚本
4. 修复脚本问题

**记录模板**：
```
【异常类型】原图未删除
【发现时间】2026-03-22 20:00
【涉及客户】[客户 ID]
【处理措施】手动删除 + 修复脚本
【责任人】[姓名]
【完成时间】2026-03-22 20:30
```

### 发现异常访问

**处理流程**：
1. 立即记录异常 IP
2. 检查是否有数据泄露
3. 封禁异常 IP
4. 通知受影响客户
5. 报告监管部门（如必要）

**记录模板**：
```
【异常类型】未授权访问
【发现时间】2026-03-22 20:00
【异常 IP】[IP 地址]
【涉及数据】[数据类型]
【处理措施】封禁 IP + 通知客户
【责任人】[姓名]
【完成时间】2026-03-22 21:00
```

### 收到客户删除请求

**处理流程**：
1. 验证客户身份
2. 删除所有相关数据
3. 记录删除操作
4. 回复客户确认

**处理时限**：7 个工作日内
