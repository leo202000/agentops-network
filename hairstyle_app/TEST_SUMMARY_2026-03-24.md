**现象**: API 返回的是文本描述，不是图片 URL

**当前 API 响应**:
```json
{

  }
}
```

**期望响应**:
```json
{
  "data": {
    "image_urls": ["https://..."],
```

1. API 端点或参数不正确
2. 需要使用正确的图生图模型
3. 请求体格式需要调整
