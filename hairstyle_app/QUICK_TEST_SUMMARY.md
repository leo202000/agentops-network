### API 并发限制

**现象**: 短时间内多次请求会触发 50430 错误（并发限制）或 100010 错误（签名验证失败）

**原因**: 即梦 API 有并发请求限制

**解决方案**:
1. ✅ 已实现重试机制（自动处理）
