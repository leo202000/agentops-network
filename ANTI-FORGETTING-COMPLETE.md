## 🎯 使用流程

### 每次会话前 (5 分钟)

```bash
# 1. 运行会话启动脚本
./start-session.sh

# 2. 查看输出
- 技能清单已更新
- 核心记忆文件已检查
- 当日日志已创建/打开
- 进行中任务已显示

# 3. 开始工作
```

### 安装技能时 (3 分钟)

```bash
# 1. 使用 skill-vetter 审查
# 2. 安装技能
# 3. 自动更新清单
./skills/update-skills-list.sh

# 4. 手动记录
vim skills/CHANGELOG.md

# 5. 记录到当日日志
vim memory/YYYY-MM-DD.md
```

### 发生重要事件时 (2 分钟)

```bash
# 1. 选择模板
cp memory/EVENT_TEMPLATE.md memory/events/event.md

# 2. 填写内容
vim memory/events/event.md

# 3. 记录到当日日志
```

### 每周审查 (15-20 分钟)

```bash
# 1. 打开审查清单
vim WEEKLY_REVIEW.md

# 2. 按步骤检查
# 3. 填写审查报告
# 4. 优化系统
```
