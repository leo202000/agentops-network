### 3. Cron 定时任务

**配置**:
```bash
# 每小时更新技能清单
0 * * * * cd /root/.openclaw/workspace && ./skills/update-skills-list.sh

# 每天早上 9 点启动会话
0 9 * * * cd /root/.openclaw/workspace && ./start-session.sh

# 每天早上 8 点监控系统
0 8 * * * cd /root/.openclaw/workspace && ./monitor-session.sh

# 每周日晚上 8 点提醒审查
0 20 * * 0 echo "记得执行每周审查！"
```

**日志文件**:
- `/var/log/openclaw-skills.log` - 技能更新日志
- `/var/log/openclaw-session.log` - 会话启动日志
- `/var/log/openclaw-monitor.log` - 监控日志

---

### 4. 监控脚本

**文件**: `monitor-session.sh`

**监控项**:
1. ✅ start-session.sh 是否存在
2. ✅ 技能清单更新频率 (>24 小时告警)
3. ✅ MEMORY.md 更新频率 (>7 天告警)
4. ✅ 当日日志是否创建
5. ✅ .env 文件权限 (应为 600)
6. ✅ 磁盘空间 (>90% 严重告警)

**告警方式**:
- 日志记录
- 邮件告警 (需配置邮箱)
- 终端输出
