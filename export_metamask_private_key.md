# 🔐 导出 MetaMask 私钥指南

## ⚠️ 安全警告

**私钥是您钱包的最高权限！**
- 拥有私钥 = 拥有钱包控制权
- 永远不要分享给任何人
- 只存储在安全的地方
- 不要截图、不要发云端

---

## 📋 导出步骤

### 方法 1：MetaMask 浏览器扩展

1. **打开 MetaMask**
   - 点击浏览器右上角的狐狸图标

2. **进入账户详情**
   - 点击账户名称（如"账户 1"）
   - 或点击三个点菜单 → 账户详情

3. **导出私钥**
   - 点击"导出私钥"按钮
   - 输入您的 MetaMask 密码
   - 点击"确认"

4. **复制私钥**
   - 私钥会显示（以 0x 开头，66 个字符）
   - 点击复制图标
   - **立即粘贴到安全地方**

5. **粘贴到配置文件**
   - 打开 `.env` 文件
   - 在 `AGC_PRIVATE_KEY=` 后面粘贴
   - 保存文件

---

### 方法 2：MetaMask 手机 App

1. **打开 MetaMask App**
2. **点击设置（齿轮图标）**
3. **选择账户**
4. **点击"导出私钥"**
5. **输入密码**
6. **复制私钥**

## 📝 私钥格式

正确的私钥格式：
```
0x1234567890abcdef1234567890abcdef1234567890abcdef1234567890abcdef
```

特征：
- 以 `0x` 开头
- 总共 66 个字符
- 包含 0-9 和 a-f

## 🔒 安全存储建议

### ✅ 推荐做法
- 存储在加密的密码管理器中
- 写在纸上，放在保险箱
- 使用硬件钱包（Ledger/Trezor）
- 仅在需要时导入

### ❌ 危险做法
- 存储在电脑明文文件中
- 发送到手机或云端
- 截图保存
- 分享给他人
- 上传到 GitHub

## 🎯 配置到挖矿脚本

### 1. 打开 .env 文件
```bash
# 编辑 .env 文件
nano .env
# 或使用其他编辑器
```

### 2. 粘贴私钥
```env
AGC_PRIVATE_KEY=0x你的私钥（66 个字符）
```

### 3. 保存文件
- 按 Ctrl+O 保存
- 按 Ctrl+X 退出（nano 编辑器）

### 4. 设置文件权限（推荐）
```bash
# 仅允许所有者读写
chmod 600 .env
```

### 5. 测试配置
```bash
python3 agentcoin_auto_mine.py
```

## ⚠️ 常见问题

### Q: 私钥丢失怎么办？
A: 如果私钥丢失且没有备份，钱包将无法恢复。务必安全备份！

### Q: 私钥被盗怎么办？
A: 立即转移资金到新钱包，旧钱包不再安全。

### Q: 可以在多个设备使用同一私钥吗？
A: 可以，但不推荐。每个设备都增加泄露风险。

### Q: 需要为每个平台创建新钱包吗？
A: 推荐。不同平台使用不同钱包，降低风险。
