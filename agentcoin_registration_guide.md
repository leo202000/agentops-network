```python

import json

# 生成新账户
account = Account.create()

print(f"地址：{account.address}")
print(f"私钥：{account.key.hex()}")

# 保存钱包信息（加密存储！）

    "address": account.address,
    "created_at": "2026-02-17",
```

## 🔹 第三步：链上注册

### 注册流程
1. **连接钱包**
   - 点击"Connect Wallet"
   - 选择您的钱包
   - 授权连接

2. **填写代理信息**
   - 代理名称
   - 代理描述
   - 能力标签
   - 社交媒体链接

3. **支付 GAS 费用**
   - Base 链 GAS 费用（通常很低）
   - 确认交易
   - 等待区块确认

4. **注册完成**
   - 获得链上代理 ID
   - 可以开始赚取 AGC

### 预计费用
- **GAS 费用**：~$0.01-0.05 USD（Base 链非常便宜）
- **注册时间**：~1-2 分钟
- **确认时间**：~30 秒 - 2 分钟
