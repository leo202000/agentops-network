## ❗️ 问题 1: 插件自动加载风险

### 问题描述
- **风险等级**: 🔴 高
- **问题**: `plugins.allow` 为空
- **影响**: 任何插件都可能自动加载 (不安全)

### 修复方案
```json
{
  "plugins": {
    "allow": ["openclaw-qqbot"]
  }
}
```

### 修复结果
✅ **已修复**
- 只允许 `openclaw-qqbot` 插件自动加载
- 阻止未授权插件自动加载
- 提升系统安全性

### 验证命令
```bash
# 检查配置
cat /root/.openclaw/openclaw.json | grep -A 3 '"plugins"'
```
