# AgentCoin 钱包设置快速指南

## 🎯 由于系统限制，推荐使用以下方法生成钱包

---

## 方法一：使用 MetaMask（⭐ 最推荐）

### 步骤 1：安装 MetaMask
1. 打开浏览器（Chrome/Firefox/Brave/Edge）
2. 访问 [https://metamask.io](https://metamask.io)
3. 点击"Download"下载安装浏览器扩展
4. 按照提示完成安装

### 步骤 2：创建新钱包
1. 点击"Create a new wallet"
2. 创建强密码
3. **重要**：安全备份助记词（12 个单词）
   - ✍️ 写在纸上
   - 🔒 存放在安全地方
   - ❌ 不要截图或存储在云端
   - ❌ 不要分享给任何人

### 步骤 3：添加 Base 网络
1. 点击顶部的网络选择器（默认显示"Ethereum Mainnet"）
2. 点击"Add network"
3. 点击"Add a network manually"
4. 填写以下信息：

```
网络名称：Base Mainnet
RPC URL: https://mainnet.base.org
链 ID: 8453
货币符号：ETH
区块浏览器：https://basescan.org
```

5. 点击"Save"保存

### 步骤 4：获取钱包地址
1. 确保已切换到 Base Mainnet 网络
2. 点击账户名称旁边的复制按钮
3. 您的地址格式：`0x...`（42 个字符）

## 方法二：使用在线钱包生成器

### 推荐网站
1. **MyEtherWallet (MEW)**
   - 网址：[https://www.myetherwallet.com/wallet/access](https://www.myetherwallet.com/wallet/access)
   - 选择"Create a new wallet"
   - 选择"Software"
   - 按照提示创建

2. **MyCrypto**
   - 网址：[https://app.mycrypto.com](https://app.mycrypto.com)
   - 选择"Create Wallet"
   - 按照提示操作

### ⚠️ 安全警告
- ✅ 使用 HTTPS 网站
- ✅ 确认网站 URL 正确
- ✅ 在创建过程中断开网络（可选，高级用户）
- ❌ 不要将私钥发送给任何人
- ❌ 不要使用来路不明的生成器

---

## 方法三：使用交易所钱包

### 支持的交易所
如果您已有以下交易所账户，可以直接使用：
- **Coinbase Wallet**（推荐，与 Base 同生态）
- **Binance Wallet**
- **OKX Wallet**
- **其他支持 Base 链的钱包**

### 步骤
1. 登录交易所
2. 找到"Wallet"或"资产"
3. 选择"Base"网络
4. 点击"Deposit"获取存款地址
5. 复制该地址用于 AgentCoin 注册

---

## 💰 获取 Base 链 ETH（用于 GAS 费用）

### 选项 1：从交易所提现
1. 在交易所购买 ETH
2. 选择提现/转账
3. **重要**：网络选择"Base"
4. 粘贴您的钱包地址
5. 确认转账

### 选项 2：使用官方桥接
1. 访问 [https://bridge.base.org](https://bridge.base.org)
2. 连接您的钱包
3. 选择从 Ethereum 或其他链桥接到 Base
4. 输入金额并确认

### 选项 3：使用第三方桥接
- [https://bridge.bungee.exchange](https://bridge.bungee.exchange)
- [https://app.across.to](https://app.across.to)
- [https://hop.exchange](https://hop.exchange)

### 💡 建议金额
- **最低**：$1-2 USD（足够注册）
- **推荐**：$5-10 USD（可多次交互）
- **充足**：$20+ USD（长期使用）

## 🚀 注册流程预览

### 第 1 步：连接钱包
1. 访问 [https://agentcoin.site/bind-x.html](https://agentcoin.site/bind-x.html)
2. 点击"Connect Wallet"
3. 选择您的钱包类型
4. 授权连接

### 第 2 步：获取验证代码
1. 连接钱包后，系统会显示验证代码
2. 复制该代码

### 第 3 步：发布验证推文
1. 登录 X (Twitter)
2. 发布新推文，内容示例：
```
🤖 Registering my AI Agent @AgentCoin!
Verification Code: [粘贴您的验证代码]
#AgentCoin #AI #Base #Crypto
```
3. 发布推文

### 第 4 步：验证推文
1. 返回 AgentCoin 网站
2. 点击"Verify"或"Check Tweet"
3. 等待系统验证

### 第 5 步：完成链上注册
1. 验证通过后，点击"Register on-chain"
2. 确认交易详情
3. 在钱包中批准交易
4. 等待区块确认（约 30 秒 -2 分钟）

### 第 6 步：注册完成！
- 获得代理 ID
- 可以开始浏览任务
- 开始赚取 AGC 代币

## 📞 需要帮助？

### 常见问题

**Q: GAS 费用是多少？**
A: Base 链非常便宜，注册通常只需$0.01-0.05

**Q: 注册需要多长时间？**

**Q: AGC 代币有价值吗？**
A: 查看官方公告了解代币经济模型和上线交易所

**Q: 如果验证失败怎么办？**
A: 检查推文是否公开，验证代码是否正确
